## GitHub repository finder


- Based on the GraphQL Github API.

### Development:
- Clone the repo.
- Once it is in your workspace, open a terminal session and then: 

```bash
$ cd frontend
$ export REACT_APP_GITHUB_API_KEY=your_github_key
$ npm install
$ npm run start
```
### Tests:
```bash
$ cd frontend
$ npm run test tests/
```
